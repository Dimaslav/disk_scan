# Сканер диска

Простое настольное приложение на Python и PyQt6 для анализа содержимого папок. Позволяет быстро просмотреть структуру каталогов, найти крупные файлы, выполнить поиск и открыть найденные файлы напрямую из интерфейса.

## Возможности

- Сканирование любой выбранной папки.
- Построение дерева каталогов.
- Подсчёт:
  - количества файлов;
  - суммарного размера каждой папки;
  - общего количества файлов;
  - общего объёма данных.
- Автоматическая классификация файлов по типам:
  - Изображения;
  - Видео;
  - Аудио;
  - Документы;
  - Архивы;
  - Код;
  - Приложения;
  - Дисковые образы;
  - Прочее.
- Поиск по:
  - имени файла;
  - полному пути;
  - категории.
- Сортировка по:
  - имени;
  - пути;
  - категории;
  - размеру;
  - дате изменения.
- Отображение прогресса во время сканирования.
- Возможность отменить сканирование.
- Открытие файла двойным щелчком.
- Работа интерфейса без зависаний благодаря использованию отдельного потока (`QThread`).

## Требования

- Python 3.10+
- PyQt6

## Установка

Клонируйте репозиторий:

```bash
git clone https://github.com/USERNAME/disk-scanner.git
cd disk-scanner
```

Установите зависимости:

```bash
pip install PyQt6
```

или

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

или

```bash
python3 main.py
```

## Использование

1. Запустите программу.
2. Выберите папку кнопкой «Обзор...» или введите путь вручную.
3. Нажмите «Сканировать».
4. После завершения:
   - выберите нужную папку в дереве слева;
   - отсортируйте список файлов по нужному столбцу;
   - используйте строку поиска для фильтрации;
   - дважды щёлкните по файлу, чтобы открыть его.

## Особенности реализации

- Используется архитектура Model/View (`QAbstractTableModel` + `QSortFilterProxyModel`).
- Сканирование выполняется в отдельном потоке.
- Предварительно вычисляются данные для быстрого поиска и фильтрации.
- Используется словарь расширений для быстрой классификации файлов.
- Дерево папок строится с накоплением статистики по размерам и количеству файлов.
